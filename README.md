# MamboTTS - 曼波文字转语音配音助手 🎧

MamboTTS 是一款专为视频创作者（如 B站/抖音小UP主）打造的 **本地大模型语音合成（TTS）桌面客户端**。

它提供了一套极其优美的深色极简图形界面（PySide6 构建），并完美集成了 **GPT-SoVITS 语音克隆大模型引擎**，支持利用您的 **本地 NVIDIA 显卡（如 RTX 5070 等）** 进行极速、高质量的“曼波风格”讲故事配音。

> 💡 **项目初心**：剪映等视频剪辑软件的“曼波讲故事”等网感音色需要购买 VIP 才能使用，本工具致力于让创作者能够通过本地显卡零成本、离线且极速地合成同样高质量的声音。

---

## ✨ 核心功能亮点

*   🎨 **高颜值现代界面**：基于 PySide6 构建的猫咪深色主题（Catppuccin 风格）极简操作窗口。
*   🚀 **本地 GPU 极速推理**：完全离线运行，秒级输出高保真音频，充分释放显卡（支持 RTX 30/40/50 系列）性能。
*   ⚙️ **一键引擎安装器 (`run_installer.bat`)**：利用国内魔搭社区极速通道，一键部署专为 NVIDIA 显卡优化预编译的 GPT-SoVITS 推理引擎，免去复杂的 Python/CUDA 编译配环境烦恼。
*   📥 **音色包一键导入 (`run_importer.bat`)**：自动扫描系统下载夹，精准识别从网盘下载的 `.pth` 和 `.ckpt` 音色权重，一键归档部署。
*   ⚡ **精准语速调节**：支持 **0.1x 到 3.0x** 语速滑块调节，引擎原生支持，变速不变调。
*   🎵 **音源下载工具 (`run_downloader.bat`)**：集成交互式下载脚本，自动拉取 B 站视频音轨并用便携版 ffmpeg 转换为无损 WAV，方便直接用于音色训练。

## 📂 项目结构说明

*   `app.py` —— 客户端 GUI 主程序。
*   `inference.py` —— 负责与本地引擎通信的客户端核心。
*   `downloader.py` / `run_downloader.bat` —— B站音频语料素材下载与处理工具。
*   `install_engine.py` / `run_installer.bat` —— 大模型推理引擎一键下载/解压部署脚本。
*   `import_weights.py` / `run_importer.bat` —— 音色包模型权重全自动导入工具。
*   `run_engine.py` / `run_local_engine.bat` —— 本地 GPU 语音合成引擎（API 服务）启动器。
*   `run.bat` —— 客户端主程序入口。

## 🚀 快速上手部署指南

### 第一步：双击启动客户端（配置环境）
1.  双击运行主目录下的 **`run.bat`**。
2.  脚本会自动为您创建虚拟环境（`.venv`）并下载安装所需的 PySide6 界面依赖库。完成配置后会自动弹出主界面。

### 第二步：一键配置本地大模型引擎
1.  双击运行主目录下的 **`run_installer.bat`**。
2.  脚本会从国内镜像源闪电下载约 3.8GB 的免安装本地大模型整合包，并在后台使用 7-Zip 工具自动解压并完成 API 路径配置。

### 第三步：一键导入您的曼波音色模型
1.  在浏览器（如 B 站 UP 主分享的网盘）下载您喜欢的“**GPT-SoVITS 曼波音色包**”（包含一个 `.pth` 文件和一个 `.ckpt` 文件）。
2.  直接双击运行项目目录下的 **`run_importer.bat`**，脚本将自动在您的“下载”文件夹中搜索这两个文件并精准移动到引擎的模型文件夹中，无需手动复制。
    *   *注：您也可以自己准备 10-20 分钟的目标音源，在 `go-webui.bat` 中通过图形界面花 20 分钟训练专属音色。*

### 第四步：极速配音
1.  双击运行 **`run_local_engine.bat`**，这会使用您的显卡启动本地 API 服务。
2.  双击运行 **`run.bat`** 打开客户端。
3.  顶部状态亮起绿色 `🟢 本地 GPU 引擎在线`。
4.  在文案框输入您的配音文字，调节您喜欢的语速（如 `1.15x`），点击 **`⚡ 合成配音`** 后点击播放，即可享受配音效果！
