# MamboTTS - 曼波文字转语音配音助手 🎧

MamboTTS 是一款专为视频创作者打造的 **本地大模型语音合成（TTS）桌面客户端**。

它提供了一套极其优美的深色极简图形界面（PySide6 构建），并完美集成了 **GPT-SoVITS 语音克隆大模型引擎**，支持利用您的 **本地 NVIDIA 显卡** 进行极速、高质量的“曼波风格”讲故事配音。

> 💡 **项目初心**：剪映等视频剪辑软件的“曼波讲故事”等网感音色需要购买 VIP 才能使用，本工具致力于让创作者能够通过本地显卡零成本、离线且极速地合成同样高质量的声音。

---

## ✨ 核心功能亮点

*   🎨 **高颜值现代界面**：基于 PySide6 构建的猫咪深色主题（Catppuccin 风格）极简操作窗口。
*   🚀 **本地 GPU 极速推理**：完全离线运行，秒级输出高保真音频，充分释放显卡（支持 RTX 30/40/50 系列）性能。
*   ⚙️ **一键引擎安装器 (`run_installer.bat`)**：利用国内魔搭社区极速通道，一键部署专为 NVIDIA 显卡优化预编译的 GPT-SoVITS 推理引擎，免去复杂的 Python/CUDA 编译配环境烦恼。
*   📦 **内置预训练曼波音色模型**：项目内置了经过高质量语料精调训练的“曼波”专属音色包，开箱即用，无需用户手动寻找音色或训练。
*   ⚡ **精准语速调节**：支持 **0.1x 到 3.0x** 语速滑块调节，引擎原生支持，变速不变调。

---

## 📂 项目结构说明

*   `app.py` —— 客户端 GUI 主程序。
*   `inference.py` —— 负责与本地引擎通信的客户端核心。
*   `models/` —— **音色模型库存放目录**。包含预训练的 `.pth` 和 `.ckpt` 文件，以及默认参考音频 `refer.wav`。
*   `install_engine.py` / `run_installer.bat` —— 大模型推理引擎一键下载/解压部署脚本。
*   `run_engine.py` / `run_local_engine.bat` —— 本地 GPU 语音合成引擎（API 服务）启动器。
*   `run.bat` —— 客户端主程序入口。

## 🚀 快速上手部署指南

为了实现最极致的“开箱即用”体验，项目已将**客户端源码与训练好的曼波模型打包整合**。您只需按照以下几步操作即可开启配音：

1.  **下载完整整合包**：
    *   前往本仓库的 **[Releases](https://github.com/Tsukimisaka/MamboTTS/releases)** 页面，下载 **`MamboTTS-v1.0.0-full.zip`**。
2.  **解压文件**：
    *   将下载好的压缩包解压到您电脑中一个**全英文路径**的文件夹下。该压缩包内已内置好了全部源码以及配好的曼波模型权重（`models/` 目录中）。
3.  **配置本地大模型引擎**：
    *   双击运行主目录下的 **`run_installer.bat`**。
    *   脚本会自动从国内镜像极速下载约 3.8GB 的免安装本地大模型推理引擎，并在后台自动完成解压和路径关联。
4.  **启动引擎服务**：
    *   双击运行 **`run_local_engine.bat`**，使用您的显卡启动本地 API 服务。
5.  **启动配音助手**：
    *   双击运行 **`run.bat`** 打开客户端。
    *   软件顶部状态亮起绿色 `🟢 本地 GPU 引擎在线`。在文案框输入您的配音文字，调节语速（如 `1.15x`），点击 **`⚡ 合成配音`** 后播放，即可开始创作！
